Tema proiect: Magazin articole vanatoare
Nume: Cristescu Teodor
Grupa de studiu: C113-A

Proiectul contine urmatoarele pagini:

1.Pagina de start(Proiect.php) in care regasim:
-Logoul site ului (pistolul din coltul stanga sus).	
-Meniu cu categorii de produse(este implementata doar o categorie- cea de accesorii (index1.php) ).
-galerie foto cu 3 imagini sugestive.
-produsele reduse expuse.
-subsol cu detalii despre site( termeni si conditii, paginile social media, contact, conditii retur).
 Butoane sugestive care ne vor duce catre alte pagini:
	-Contact ->contact.php
	-Autentificare -> register.php
	-Login	->login.php
2.Pagina contact.php prezinta detaliile de contact ale magazinului alaturi de o poza sugestiva.
3.Pagina register contine campurile aferente inregistrarii pe site.
4.Pagina login permite logarea pe site cu ajutorul username-ului si a parolei printr-o sesiune.

Dupa ce se logheaza, un untilizator poate accesa urmatoarele 2 pagini cu ajutorul butoanelor din dreapta sus:

4.Editare profil- unde poate sa editeze datele sale din baza de date.
5.Cosul de cumparaturi(cart.php) unde poate vedea produsele adaugate in cos alaturi de pret, imagine, cantitate. De acolo se poate elimina un produs din cos sau se poate modifica cantitatea acestuia.
Mai departe avem:
6.Pagina index1.php permite vizualizarea produselor cu pretul si imaginea lor  din o anumita categorie alaturi de butonul adauga in cos.
8.Pagina product.php este cea care realizeaza vizualizarea detaliilor unui produs (pagina se acceseaza dand click pe un produs).
			
Pentru Administrator avem pagina pagadmin.php unde vor fi afisati toti clientii
	-Sunt disponibile stergerea si editarea detaliilor clientilor fie dupa un Id(->Editare clienti, ->Stergere clienti), fie direct din tabelul unde sunt afisati dand click pe iconitele sugestive.

In orice pagina dand click pe logo-ul site ului utilizatorul va fi directionat catre pagina de start.

De asemenea in proiect mai gasim folderele img(unde sunt stocate imaginile) si css ce contine (style.css).

Am atasat si exportul bazei de date pentru a putea fi realizata navigarea in paginile site ului. Acolo se gasesc username-uri si parole.
